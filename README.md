# AI Chatbot Application

A responsive Flask-based AI chatbot application with data visualization capabilities.

## Features

- 🤖 AI-powered chat interface
- 📊 Data visualization (bar charts, pie charts)
- 📱 Responsive design (600px-1400px)
- 🔐 User authentication
- 💬 Multiple chat interfaces
- 📁 File upload and data analysis

## Quick Start

1. Clone the repository
2. Install dependencies: npm i
3. Run the application: node backend/server.js
4. Open http://localhost:5000

## Project Structure

├── requirements.txt  
├── templates/         # HTML templates
├── static/           # CSS, JavaScript, images
└── instance/         # Database files

## Deployment

Ready for cloud deployment on:
- Render
- Railway
- AWS Elastic Beanstalk
